Автоматическая генерация тематического контента

Команда Enot Ads Network создала собственную IT-инфраструктуру, позволяющую с помощью нейросетей в автоматическом режиме формировать текстовый контент и медиа-контент на множество тематик с соблюдением норм законодательства РФ. С помощью неё десятки каналов гарантированно обеспечены качественно написанным, структурированным и логично связанным контентом. Для реализации этой системы мы используем современные инструменты разработки: GPT-4, LangChain, FastAPI, PostgreSQL, Docker, GitHub Actions, Telegram Bot API. Благодаря такому подходу время, затрачиваемое на создание каждого поста, сократилось примерно в 10 раз.

Автоматизированное управление Telegram-каналами

Используя возможности Python и фреймворка FastAPI, была разработана система автоматизированного размещения рекламных постов с контентом по заданному расписанию. За взаимодействие с Telegram API отвечает библиотека aiogram, а за хранение информации о клиентах, постах и результатах размещения — база данных PostgreSQL. Публикация контента реализована через Celery и Redis, что позволяет точно управлять временем размещения и выполнять задачи в фоновом режиме. Особенностью системы является анализ трафика на органичность и противодействие «накрутки» просмотров рекламно-информационного материала различными ботами, что обеспечивает заказчику эффективную трату средств на рекламу.


Автоматизированный анализ и отчетность

С целью доказательного отнесения затрат клиентов на рекламу платформа Enot Ads Network позволяет формировать фото/видео-фактуры размещения рекламы в telegram-каналах без участия человеческого ресурса, что понижает вероятность ошибок при фиксации.Для реализации данной автоматизации были использованы следующие инструменты: Pandas, PostgreSQL, Pillow и OpenCV, Jinja2 + WeasyPrin, Celery и Redis,SMTP и Telegram Bot API. Это снизило вероятность ошибок, повысило надежность бизнес-модели и удовлетворенность клиентов.
